# 学习笔记审校 Prompt（用于 Claude Code）

请作为审校助手，仔细检查我下面提供的学习笔记。
你的任务：
1. 找出笔记中**事实性错误**，以及**表述不清晰、容易产生歧义**的地方。
2. 对于每一个需要修改的位置，直接在原文处进行改动，**不要单独列表或总结**。
3. 保持笔记原有的整体结构、Markdown 格式、代码块、公式等内容完全不变，只替换有问题的文字。

## 修改格式（必须严格遵守）

使用以下格式进行修改，不要添加任何额外说明或引言：

- 用 Markdown 删除线 `~~` 把**我原稿中有问题或不当的表述**划掉。
- 紧接着用 HTML 红色字体标注出**更准确、更清晰的表达**，格式为：  
  `<span style="color:red">修正后的内容</span>`
- 如果一句话里既有错误也有表述不清，可以合并处理，一次性给出完整的修正。
- 没有问题的部分请原样保留，不要改动。

## 修改示例

原始内容：
> Python 的列表是不可变的数据结构。

审校后输出：
> Python 的列表是~~不可变的~~<span style="color:red">可变的</span>数据结构。

## 注意事项

- 不要评价我的原笔记，不要输出“这里有问题”、“改得好”等无关文字。
- 笔记中可能包含代码块和数学公式，只检查和修改中文叙述部分，**不要改动代码和公式本身**。
- 如果笔记很长需要分段输出，请每次处理一个完整段落，直到全部完成。
