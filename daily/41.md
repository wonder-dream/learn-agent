### 2. Self-Attention 公式 + 代码 (1.5h)

```
Attention(Q, K, V) = softmax(QK^T / √d_k) × V

为什么要除以 √d_k？
→ 防止点积过大导致 softmax 梯度消失
```

手写代码（面试可能让写）：

```python
import torch
import torch.nn.functional as F

def self_attention(X, W_Q, W_K, W_V):
    Q = X @ W_Q  # (batch, seq_len, d_model)
    K = X @ W_K
    V = X @ W_V

    d_k = Q.size(-1)
    scores = Q @ K.transpose(-2, -1) / math.sqrt(d_k)  # 缩放
    attn_weights = F.softmax(scores, dim=-1)            # 归一化
    output = attn_weights @ V                           # 加权求和
    return output
```

### 3. MHA / MQA / GQA (1h)

| 类型 | K V heads | 用途 |
|------|-----------|------|
| MHA (Multi-Head) | Q 几个头 K V 几个头 | 原始 Transformer |
| MQA (Multi-Query) | K V 只有 1 个头 | 推理加速（减少 KV cache） |
| GQA (Grouped-Query) | Q 几个头，K V 分 N 组共享 | DeepSeek-v4 / Llama 常用 |

### 4. 位置编码 (1h)

- **Sinusoidal**: `PE(pos, 2i) = sin(pos / 10000^(2i/d))`, 固定不可学习
- **可学习位置编码**: GPT 系列
- **RoPE**: 旋转位置编码，通过旋转矩阵把位置信息注入 Q 和 K。Llama/DeepSeek 均使用

### 5. Tokenization (0.5h)

- BPE (Byte-Pair Encoding)：GPT 系
- WordPiece：BERT
- SentencePiece：T5 / Llama
- 面试常问：中文 tokenization 的挑战
