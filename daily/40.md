## 回溯模板

```python
def backtrack(path, choices):
    if 满足结束条件:
        result.append(path[:])  # 注意拷贝！
        return

    for choice in choices:
        if 剪枝条件:
            continue
        path.append(choice)     # 做选择
        backtrack(path, new_choices)  # 递归
        path.pop()              # 撤销选择
```

## 二分模板

```python
# 找目标值（标准版）
l, r = 0, len(nums) - 1
while l <= r:
    m = (l + r) // 2
    if nums[m] == target: return m
    elif nums[m] < target: l = m + 1
    else: r = m - 1

# 找左边界
while l < r:
    m = (l + r) // 2
    if nums[m] >= target: r = m
    else: l = m + 1

# 找右边界
while l < r:
    m = (l + r + 1) // 2
    if nums[m] <= target: l = m
    else: r = m - 1
```
