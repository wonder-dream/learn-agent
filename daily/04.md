### 1. async/await 基础 (1h)

```python
import asyncio

# 普通函数 → 协程函数（加 async）
async def fetch_data(url: str) -> str:
    print(f"Fetching {url}...")
    await asyncio.sleep(1)   # 模拟 IO 等待，不是 time.sleep！
    return f"Data from {url}"

# 运行协程
async def main():
    result = await fetch_data("http://example.com")
    print(result)

asyncio.run(main())
```

**关键理解**：`await` 意思是"我先让出 CPU，你完事了叫我"——不是阻塞等待，而是把时间片给其他协程。

### 2. 并发执行 (1.5h)

```python
import asyncio
import time

async def fetch(url: str) -> str:
    await asyncio.sleep(1)
    return f"Done: {url}"

async def main():
    # 并发——总共约 1 秒
    start = time.time()
    results = await asyncio.gather(
        fetch("url1"),
        fetch("url2"),
        fetch("url3"),
    )
    print(f"并发耗时: {time.time() - start:.2f}s")  # ~1s

    # 对比：顺序执行
    start = time.time()
    for u in ["url1", "url2", "url3"]:
        await fetch(u)
    print(f"顺序耗时: {time.time() - start:.2f}s")  # ~3s

asyncio.run(main())
```

### 3. httpx 实战 (1h)


```python
import httpx
import asyncio

async def fetch_all(urls: list[str]) -> list[dict]:
    async with httpx.AsyncClient(timeout=10) as client:
        tasks = [client.get(url) for url in urls]
        responses = await asyncio.gather(*tasks)
        return [r.json() for r in responses]

# 跑一个真正的 API 请求
urls = ["https://api.github.com"] * 3
results = asyncio.run(fetch_all(urls))
```
