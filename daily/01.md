## 具体任务

### 1. Type Hints 基础 (1h)
- 阅读 Python 官方 typing 文档的常用部分
- 练习：给一个已有的 Python 函数补全类型注解
- 掌握：`Optional[str]`, `Union[int, float]`, `list[dict]`, `Literal["a", "b"]`

### 2. pydantic v2 入门 (1.5h)
- 安装 pydantic: `pip install pydantic`
- 学习 BaseModel 的基本用法
- 学习 Field 的常用参数：default, ge, le, min_length, max_length, description
- 了解 model_validate() 和 model_dump()

### 3. 练习 (1h)
- 定义以下模型：
  - `ReviewRequest`: file_path, file_content, language, review_level
  - `ReviewComment`: file_path, line_number, severity, category, message, suggestion
  - `ReviewResponse`: request_id, comments, summary, cost
- 写一个简单的测试脚本验证数据校验

### 4. 输出 (0.5h)
- 整理今天学到的内容到 `notes/engineering/python-types.md`
- 至少包含：核心概念、代码示例、常见坑
