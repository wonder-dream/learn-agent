### 1. 理解 ReAct 循环 (0.5h)

```
┌──────────────────────────────────────────────────┐
│                  ReAct Agent Loop                  │
│                                                    │
│   User Input                                       │
│       ↓                                            │
│   ┌─────────┐     ┌──────────┐     ┌──────────┐  │
│   │  THINK  │ ──→ │   ACT    │ ──→ │ OBSERVE  │  │
│   │ (LLM)   │ ←── │ (tool)   │     │ (result) │  │
│   └─────────┘     └──────────┘     └──────────┘  │
│       │                                            │
│       │ (no more tools needed)                     │
│       ↓                                            │
│   Final Answer                                    │
└──────────────────────────────────────────────────┘
```

LLM 不是一次性生成答案，而是：
1. **Think**：分析当前状态，决定下一步
2. **Act**：如果觉得需要更多信息 → 调用工具
3. **Observe**：接收工具结果
4. 回到 Think，直到 LLM 认为足够了 → 输出最终答案

```python
        "function": {

            "description": "记录一个代码问题",
            "parameters": {
                "type": "object",
                "properties": {
                    "line": {"type": "integer", "minimum": 1},
                    "severity": {"enum": ["info", "warning", "error"]},
                    "category": {"enum": ["style", "naming", "logic", "security"]},
                    "message": {"type": "string"}
                },
                "required": ["line", "severity", "category", "message"]
            }
        }
    }
]

def parse_ast(code: str) -> dict:
    """工具实现：解析 Python AST"""
    import ast
    tree = ast.parse(code)
    functions = [node.name for node in ast.walk(tree) if isinstance(node, ast.FunctionDef)]
    classes = [node.name for node in ast.walk(tree) if isinstance(node, ast.ClassDef)]
    variables = [node.id for node in ast.walk(tree) if isinstance(node, ast.Name) and isinstance(node.ctx, ast.Store)]
    return {"functions": functions, "classes": classes, "variables": variables}

TOOL_MAP = {
    "parse_ast": parse_ast,
    "report_issue": lambda **kwargs: kwargs,  # 简单回传
}

class SimpleAgent:
    def __init__(self, llm: LLMClient, max_turns: int = 10):
        self.llm = llm
        self.max_turns = max_turns
        self.tools = TOOLS

    async def run(self, task: str, system_prompt: str) -> dict:
        messages = [
            {"role": "system", "content": system_prompt},
            {"role": "user", "content": task},
        ]

        for turn in range(self.max_turns):
            response = await self.llm.chat(
                messages=messages,
                temperature=0.1,
                # 注意：这里 llm_client 需要支持 tools 参数
            )

            # 简化版：检查 LLM 是否输出 tool_call
            # (实际需要根据 API 返回的 tool_calls 判断)

            # 如果 LLM 认为审查完成，跳出
            if "FINAL_REVIEW" in response.content:
                break

        return {"turns": turn + 1, "final": response.content}
```
