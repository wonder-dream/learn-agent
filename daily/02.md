### 1. lambda + 高阶函数 (1h)

```python
# lambda：一行函数，不要在 lambda 里写复杂逻辑
square = lambda x: x**2       # 等价于 def square(x): return x**2

# sorted 的 key 参数——面试极高频
users = [("Alice", 25), ("Bob", 30), ("Charlie", 20)]
users.sort(key=lambda u: u[1])   # 按年龄排序

# map/filter——列表推导式通常更可读，但你要能看懂
nums = [1, 2, 3, 4, 5]
list(map(str, nums))             # ['1', '2', '3', '4', '5']
list(filter(lambda x: x>2, nums)) # [3, 4, 5]

# 推荐写法（比上面更 Pythonic）
[str(x) for x in nums]           # 等价于 map
[x for x in nums if x > 2]       # 等价于 filter
```

### 2. 解包 unpacking (1h)

```python
# 多重赋值
a, b = 1, 2
a, b = b, a                    # 一行交换！Java 要三行

# 星号解包——这个在 Agent 代码里大量出现
first, *rest = [1, 2, 3, 4]   # first=1, rest=[2,3,4]
first, *middle, last = range(10)

# *args 和 **kwargs——函数设计基石
def log(level, *messages, **tags):
    for msg in messages:
        print(f"[{level}] {msg} | tags={tags}")

log("ERROR", "timeout", "retry failed", service="agent", retry=3)
# [ERROR] timeout | tags={'service': 'agent', 'retry': 3}
# [ERROR] retry failed | tags={'service': 'agent', 'retry': 3}

# 展开列表/字典
params = {"host": "0.0.0.0", "port": 8000}
server = Server(**params)      # 相当于 Server(host="0.0.0.0", port=8000)
```

### 3. enumerate + zip (0.5h)

```python
# enumerate：同时拿到索引和值
for i, line in enumerate(lines):
    print(f"L{i}: {line}")

# zip：同时遍历多个序列
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 20]
for name, age in zip(names, ages):
    print(f"{name} is {age}")
```

### 4. 装饰器入门 (1h)

```python
# 装饰器就是把函数作为参数，返回一个新函数
# 看懂这行就行，自己写装饰器是进阶内容：
def log_call(func):
    def wrapper(*args, **kwargs):
        print(f"Calling {func.__name__}")
        result = func(*args, **kwargs)
        print(f"{func.__name__} returned {result}")
        return result
    return wrapper

@log_call
def add(a, b):
    return a + b

add(1, 2)
# Calling add
# add returned 3
```

了解 FastAPI 中无处不在的装饰器：`@app.get()`, `@app.post()`, `@router.middleware()`

### 5. with 语句 + 文件读写 (0.5h)

```python
# with 自动关闭资源，不用写 finally close()
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()

# 写文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("hello\n")

# 逐行读（大文件）
with open("big.txt") as f:
    for line in f:        # 不会一次性加载全部内容
        process(line)
```
