### 1. AST 基础 (1h)

```python
import ast

code = """
import os

def read_file(path: str) -> str:
    if not os.path.exists(path):
        raise FileNotFoundError(f"No such file: {path}")
    with open(path, 'r') as f:
        return f.read()

class Config:
    def __init__(self, debug: bool = False):
        self.debug = debug
"""

tree = ast.parse(code)
print(ast.dump(tree, indent=2))  # 看完整 AST 结构
```

**常用节点类型**：
| 节点 | 含义 | 关键属性 |
|------|------|----------|
| `ast.FunctionDef` | 函数定义 | name, args, body, decorator_list, lineno |
| `ast.ClassDef` | 类定义 | name, bases, body, lineno |
| `ast.Import` / `ast.ImportFrom` | import 语句 | names, module |
| `ast.Assign` | 赋值语句 | targets, value, lineno |
| `ast.Call` | 函数调用 | func, args, keywords |
| `ast.If` / `ast.For` / `ast.While` | 控制流 | test, body, orelse |
| `ast.Try` | 异常处理 | body, handlers, orelse, finalbody |
| `ast.Name` | 变量名 | id, ctx (Store/Load) |
| `ast.Attribute` | 属性访问 | value, attr |

### 2. ast.walk() vs ast.NodeVisitor (1h)

**walk**——简单粗暴，遍历所有节点：
```python
for node in ast.walk(tree):
    if isinstance(node, ast.FunctionDef):
        print(f"Function: {node.name} at line {node.lineno}")
    elif isinstance(node, ast.ClassDef):
        print(f"Class: {node.name} at line {node.lineno}")
```

**NodeVisitor**——更结构化，适合复杂分析：
```python
class CodeAnalyzer(ast.NodeVisitor):
    def __init__(self):
        self.functions = []
        self.classes = []
        self.imports = []
        self.max_nested_depth = 0
        self._current_depth = 0

    def visit_FunctionDef(self, node):
        self.functions.append({
            "name": node.name,
            "line": node.lineno,
            "arg_count": len(node.args.args),
            "body_length": len(node.body),
            "has_docstring": (
                isinstance(node.body[0], ast.Expr) and
                isinstance(node.body[0].value, ast.Constant) and
                isinstance(node.body[0].value.value, str)
            ) if node.body else False,
        })
        self._current_depth += 1
        self.max_nested_depth = max(self.max_nested_depth, self._current_depth)
        self.generic_visit(node)  # 继续遍历子节点
        self._current_depth -= 1

    def visit_ClassDef(self, node):
        self.classes.append({"name": node.name, "line": node.lineno})
        self.generic_visit(node)

    def visit_Import(self, node):
        for alias in node.names:
            self.imports.append({"module": alias.name, "alias": alias.asname})
```

**关键**：`self.generic_visit(node)` 是继续深入遍历子节点——不调这句遍历就停在当前节点。
