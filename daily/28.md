## Architecture

```
Source Code
    │
    ▼
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│ AST Parser  │ ──→ │ Candidate    │ ──→ │ LLM Deep    │
│ + Static    │     │ Extractor    │     │ Analysis    │
│ Rules       │     │              │     │             │
└─────────────┘     └──────────────┘     └─────────────┘
    │                                             │
    └───────────────── Merge ─────────────────────┘
                            │
                            ▼
                   ┌───────────────┐
                   │ Structured    │
                   │ Review Report │
                   │ (JSON/MD)     │
                   └───────────────┘
```

## Features

- **11 static analysis rules**: naming conventions, code style, cyclomatic complexity, nesting depth
- **4 LLM deep analysis types**: null pointer risks, resource leaks, broad except, loop logic bugs
- **LangGraph orchestration**: state graph with checkpoint for resumable reviews
- **3 output formats**: colored terminal, JSON, Markdown
- **REST API**: async task submission + polling
- **Structured logging** + rate limiting + token cost tracking

## Quick Start

```bash
git clone <this-repo>
cd code-review-agent
pip install -e ".[dev]"
cp .env.example .env  # add your DEEPSEEK_API_KEY
```

### CLI

```bash
python cli.py review path/to/file.py --level medium
python cli.py review path/to/file.py --format json
python cli.py review path/to/file.py --format markdown > review.md
```

### API

```bash
uvicorn src.server:app --reload
# POST /review → GET /review/{id}
```

## Tech Stack

- **LLM**: DeepSeek-v4-pro
- **Orchestration**: LangGraph (StateGraph + Checkpoint)
- **Static Analysis**: Python `ast` module
- **API**: FastAPI + asyncio
- **Testing**: pytest + pytest-asyncio + pytest-mock

## Project Structure

```
src/
├── server.py          # FastAPI app
├── graph.py           # LangGraph review pipeline
├── pipeline.py        # Linear pipeline (pre-LangGraph)
├── static_analyzer.py # AST-based analysis entry
├── rules/             # 11 static rules (style, naming, complexity)
├── analyzers/         # LLM deep analysis (candidate extraction + logic check)
├── tools/             # AST tools + retry/timeout utilities
├── llm_client.py      # DeepSeek API wrapper
├── schemas.py         # Pydantic data models
├── prompts/           # System prompts + templates
├── config.py          # Settings management
├── logging.py         # Structured JSON logging
└── rate_limiter.py    # Token bucket rate limiter
```
